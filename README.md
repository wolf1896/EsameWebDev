# Recipe Discovery & Meal Planner

## Project Overview
This project is an interactive, dynamic web application developed as a Final Exam submission for EPICODE. It is designed to help users search for food recipes and organize their meal plans. The app retrieves real-world data asynchronously from a public API, ensuring a dynamic and engaging user experience. 

## Features Included
* **Authentication State**: Includes a simulated login system that protects application routes and alters the UI depending on user status.
* **Dynamic Data Fetching**: Integrates with `TheMealDB` API using the Fetch API to populate recipe searches dynamically.
* **DOM Manipulation**: Creates, updates, and removes HTML elements interactively based on user inputs and fetched data.
* **Form Validation**: Multiple forms (Login and Search) ensure correct data formats before simulating submission.
* **Responsive Layout**: Built using Flexbox, guaranteeing that the application is fully functional and visually consistent across both desktop and mobile devices.

## Technologies Used
* **HTML5**: Semantic tags for structured, accessible content.
* **CSS3**: Flexbox layout, media queries for responsiveness, and subtle state animations.
* **JavaScript (Vanilla)**: Event handling, `localStorage` management, DOM manipulation, and asynchronous network requests.

## How to Run the Application
1. Download or clone this repository to your local machine.
2. Unzip the project folder if downloaded as a `.zip` file.
3. Open the `login.html` file in your preferred modern web browser (e.g., Chrome, Firefox, Edge).
4. Enter any valid username (minimum 3 characters) and password (minimum 6 characters) to access the application.
5. Use the navigation bar to move between the Home, Planner, and Profile views.
